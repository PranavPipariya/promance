# CodeRabbit Setup Guide

CodeRabbit is an AI-powered code review tool that will help us qualify for the **Captain Code Award ($1,000)**.

## Installation Steps

1. **Install CodeRabbit GitHub App**
   - Go to https://github.com/apps/coderabbitai
   - Click "Install" or "Configure"
   - Select the repository: `PranavPipariya/promance`
   - Grant the necessary permissions

2. **Verify Installation**
   - Go to your repository settings
   - Navigate to "Integrations" → "GitHub Apps"
   - Confirm CodeRabbit is listed and active

3. **Test CodeRabbit**
   - Create a new branch and make a small change
   - Open a Pull Request
   - CodeRabbit will automatically review your PR with:
     - Code quality suggestions
     - Bug detection
     - Performance improvements
     - Documentation recommendations
     - Best practices

## What CodeRabbit Does

- **Automated PR Reviews**: Reviews every pull request automatically
- **Code Quality**: Suggests improvements and best practices
- **Documentation**: Recommends adding docs where needed
- **Security**: Identifies potential security issues
- **Performance**: Suggests optimizations

## For the Hackathon

The Captain Code Award requires:
- ✅ CodeRabbit integration visible in PRs
- ✅ Code quality improvements based on CodeRabbit feedback
- ✅ Documentation improvements

CodeRabbit will automatically provide all of this!
